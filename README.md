# Hijinks

Experimental JavaScript engine running on the [Hivm](http://hivm.org/) virtual machine.

## Installation

Add this line to your application's Gemfile:

    gem 'hijinks'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install hijinks
